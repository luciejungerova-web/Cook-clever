# 🍳 Cook Clever

Chytré plánování jídel, nákupní seznamy a správa skladových zásob v jedné mobilní aplikaci.

## 🚀 Nasazení na GitHub Pages

### Krok 1: Vytvoř GitHub repozitář
1. Jdi na [github.com](https://github.com) a přihlaš se
2. Klikni na **+** (vpravo nahoře) → **New repository**
3. Pojmenuj repozitář: `cook-clever` (nebo libovolný název)
4. Nastav jako **Public**
5. Klikni **Create repository**

### Krok 2: Nahraj soubory
Máš 2 možnosti:

#### A) Přes webové rozhraní (nejjednodušší):
1. V repozitáři klikni **uploading an existing file**
2. Přetáhni všechny soubory z této složky:
   - `index.html`
   - `cook-clever.html`
   - `manifest.json`
   - `README.md`
3. Klikni **Commit changes**

#### B) Přes Git (pokud máš nainstalovaný):
```bash
git clone https://github.com/TVOJE-JMENO/cook-clever.git
cd cook-clever
# Zkopíruj sem všechny soubory
git add .
git commit -m "Initial commit"
git push
```

### Krok 3: Aktivuj GitHub Pages
1. V repozitáři jdi na **Settings** (nahoře)
2. V levém menu klikni **Pages**
3. Pod **Source** vyber:
   - Branch: **main** (nebo **master**)
   - Folder: **/ (root)**
4. Klikni **Save**
5. Počkej 1-2 minuty

### Krok 4: Hotovo! 🎉
GitHub ti ukáže URL (podobné):
```
https://TVOJE-JMENO.github.io/cook-clever/
```

Otevři ji na mobilu a nainstaluj jako aplikaci!

## 📱 Instalace na mobil

### Android (Chrome):
1. Otevři URL v Chrome
2. Menu ⋮ → **"Přidat na plochu"**
3. Aplikace se objeví na ploše

### iOS (Safari):
1. Otevři URL v Safari
2. Tlačítko **Sdílet** 📤
3. **"Přidat na plochu"**

## ✨ Funkce aplikace

- 📅 **Plán týdne** - naplánuj jídla na Po-Ne
- 📖 **82 receptů** - český obsah, detailní postup
- 🛍 **Smart nákup** - automaticky z plánu, vynechává zásoby
- 🏠 **Zásobník** - 40+ skladových položek
- 🧊 **Lednička** - najdi recepty podle toho co máš doma
- 📤 **Sdílení** - pošli jídelníček + nákup rodině
- 💾 **Offline** - vše funguje bez internetu

## 🔧 Aktualizace aplikace

Když chceš změnit recepty nebo funkce:
1. Uprav soubor `cook-clever.html`
2. Nahraj ho znovu do repozitáře (přepiš starý)
3. Za pár minut se změny projeví na URL
